#Lazagne

On Windows machines, the tool "Lazagne" is helpful for credential hunting as it will run through the system searching different common locations and searching for strings that match things like Password, Username, etc. 

#findstr

On Windows machines, the command "findstr" can be used to find a file based on the string that you give the command and file types that you choose. There are other options that can be useful that go with this as well.
